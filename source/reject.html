Array#reject(iterator, [context])

Return the group of elements for which the return value of the iterator is false.

Usage

Parameters

  • iterator - The iterator receives each element in turn as the first agument.
  • [context] - Optional context parameter to be used as `this` when calling the iterator function.

Returns

Type: Array

An array containing the elements for which the iterator returned false.
