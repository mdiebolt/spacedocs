Number#odd()

Returns true if this number is odd (has remainder of 1 when divided by 2).

Usage

2.odd()
            # => false
            
            3.odd()
            # => true
            
            0.odd()
            # => false

Returns

Type: Boolean

true if this number is an odd integer, false otherwise.

Source

Toggle 
Number.prototype.odd = function() {
            if (this > 0) {
              return this % 2 === 1;
            } else {
              return this % 2 === -1;
            }
          };
