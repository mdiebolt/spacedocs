Array#invoke(method, Optional)

Invoke the named method on each element in the array
and return a new array containing the results of the invocation.

Usage

[1.1, 2.2, 3.3, 4.4].invoke("floor")
            # => [1, 2, 3, 4]
            
            ['hello', 'world', 'cool!'].invoke('substring', 0, 3)
            # => ['hel', 'wor', 'coo']

Parameters

  • method - The name of the method to invoke.
  • Optional - arguments to pass to the method being invoked.

Returns

Type: Array

A new array containing the results of invoking the named method on each element.

Array.prototype.invoke = function() {
            var args, method;
            method = arguments[0], args = 2 <= arguments.length ? __slice.call(arguments, 1) : [];
            return this.map(function(element) {
