Objectextend(target)

Merges properties from sources into target with overiding.
Last in covers earlier properties.

Usage

I =
              a: 1
              b: 2
              c: 3
            
            Object.extend I,
              c: 6
              d: 4
            
            I # => {a: 1, b:2, c:6, d: 4}

Parameters

  • target - The object to merge the properties into.

Returns

Type: Object
