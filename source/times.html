Number#times(iterator, [context])

Calls iterator the specified number of times, passing in the number of the
current iteration as a parameter: 0 on first call, 1 on the second call, etc.

Usage

output = []
            
            5.times (n) ->
              output.push(n)
            
            output
            # => [0, 1, 2, 3, 4]

Parameters

  • iterator - The iterator takes a single parameter, the number of the current iteration.
  • [context] - The optional context parameter specifies an object to treat as `this` in the iterator block.

Returns

Type: Number

The number of times the iterator was called.
