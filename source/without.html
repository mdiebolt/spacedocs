Array#without(values)

Return the group of elements that are not in the passed in set.

Usage

[1, 2, 3, 4].without ([2, 3])
            # => [1, 4]

Parameters

  • values - List of elements to exclude.

Returns

Type: Array

An array containing the elements that are not passed in.
