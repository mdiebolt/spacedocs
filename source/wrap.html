Array#wrap(start, [length])

Pretend the array is a circle and grab a new array containing length elements.
If length is not given return the element at start, again assuming the array
is a circle.

Usage

[1, 2, 3].wrap(-1)
            # => 3
            
            [1, 2, 3].wrap(6)
            # => 1
            
            ["l", "o", "o", "p"].wrap(0, 16)
            # => ["l", "o", "o", "p", "l", "o", "o", "p", "l", "o", "o", "p", "l", "o", "o", "p"]

Parameters

  • start - The index to start wrapping at, or the index of the sole element to return if no length is given.
  • [length] - Optional length determines how long result array should be.

Returns

Type: Object

or {Array} The element at start mod array.length, or an array of length elements, starting from start and wrapping.

Array.prototype.wrap = function(start, length) {
            var end, i, result;
            if (length != null) {
              end = start + length;
              i = start;
              result = [];
              while (i++ < end) {
                result.push(this[i.mod(this.length)]);
              }
              return result;
            } else {
              return this[start.mod(this.length)];
            }
          };
