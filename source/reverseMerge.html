ObjectreverseMerge(target)

Merges properties from objects into target without overiding.
First come, first served.

Usage

I =
              a: 1
              b: 2
              c: 3
            
            Object.reverseMerge I,
              c: 6
              d: 4
            
            I # => {a: 1, b:2, c:3, d: 4}

Parameters

  • target - The object to merge the properties into.

Returns

Type: Object
