Number#snap(resolution)

Returns the the nearest grid resolution less than or equal to the number.

Usage

7.snap(8)
            # => 0
            
            4.snap(8)
            # => 0
            
            12.snap(8)
            # => 8

Parameters

  • resolution - The grid resolution to snap to.

Returns

Type: Number

The nearest multiple of resolution lower than the number.
