Array#eachPair(iterator, [context])

Call the given iterator once for each pair of objects in the array.

Usage

[1, 2, 3, 4].eachPair (a, b) ->
              # 1, 2
              # 1, 3
              # 1, 4
              # 2, 3
              # 2, 4
              # 3, 4

Parameters

  • iterator - Function to be called once for each pair of elements in the array.
  • [context] - Optional context parameter to be used as `this` when calling the iterator function.

Returns
