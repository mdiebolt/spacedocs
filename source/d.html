Number#d(sides)

The mathematical d operator. Useful for simulating dice rolls.

Usage

# Simulates rolling 5 6-sided dice
            5.d(6)
            
            # Simulates rolling 3 20-sided dice
            3.d(20)

Parameters

  • sides - Number of sides on the simulated die

Returns

Type: Number

Sum of rolling `this` many `sides`-sided dice

Number.prototype.d = function(sides) {
            var sum;
            sum = 0;
            this.times(function() {
              return sum += rand(sides) + 1;
            });
