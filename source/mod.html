Number#mod(base)

A mod method useful for array wrapping. The range of the function is
constrained to remain in bounds of array indices.

Usage

(-1).mod(5)
            # => 4

Parameters

  • base -

Returns

Type: Number

An integer between 0 and (base - 1) if base is positive.

            var result;
            result = this % base;
            if (result < 0 && base > 0) result += base;
            return result;
