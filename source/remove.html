Array#remove(object)

Remove the first occurrence of the given object from the array if it is
present. The array is modified in place.

Usage

a = [1, 1, "a", "b"]
            a.remove(1)
            # => 1
            
            a
            # => [1, "a", "b"]

Parameters

  • object - The object to remove from the array if present.

Returns

Type: Object

The removed object if present otherwise undefined.

Array.prototype.remove = function(object) {
            var index;
            index = this.indexOf(object);
            if (index >= 0) {
              return this.splice(index, 1)[0];
            } else {
              return;
            }
          };
